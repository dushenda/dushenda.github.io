## Ubuntu
设置如下的程序
```C++
#include <iostream>
#include <string.h>
using namespace std;

void crash_test()
{
        char *str =0;
        strcpy(str,"test"); // segment fault
}
int main()
{
        cout << "crash test" << endl;
        crash_test();
        return 0;
}
```

### 查看coredump是否打开
查看命令：`ulimit -c`
打开命令：`ulimit -c unlimited`
0：关闭
unlimited：打开
